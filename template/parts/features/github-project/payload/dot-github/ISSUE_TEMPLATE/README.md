# .github/ISSUE_TEMPLATE

## 1. 概要

GitHub Issue テンプレートを管理するディレクトリです。

## 2. 責任

- バグ報告テンプレート（`bug_report.yml`）
- 機能追加テンプレート（`feature_request.yml`）
- タスクテンプレート（`task.yml`）
- テンプレート設定（`config.yml`）

## 3. 責任外

- Pull Request テンプレート（`../ PULL_REQUEST_TEMPLATE.md` が担当）
- GitHub Actions ワークフロー（`../workflows/` が担当）
