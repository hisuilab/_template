# docs/domain/rules

## 1. 概要

ビジネスルールと不変条件を一覧するディレクトリです。
領域が増えた場合は `<domain-area>.md` を追加して分割できます。

## 2. 責任

- 集約をまたぐビジネスルールの列挙
- 不変条件・制約・業務上の判断基準

## 3. 責任外

- ルールの実装詳細（`src/domain/` が担当）
- 画面バリデーション（インターフェース層が担当）

---

## ルール一覧

### （集約名 / 領域名）

- （ルールをここに追加していく）

---

## ルールの書き方

```markdown
### <集約名 / 領域名>

- <何が> <どういう条件で> <どうなる>
```
