# docs/milestones

## 1. 概要

マイルストーン単位の詳細設計・完了条件を管理するディレクトリです。

## 2. 責任

- 保持するもの: マイルストーン文書（`m{N}-slug.md`）

## 3. 責任外

- 要件・アーキテクチャの上位方針（`docs/draft/project-direction.md` が持ちます）

## 4. 凡例

| status | 意味 |
| --- | --- |
| `not_started` | 未着手 |
| `in_progress` | 進行中 |
| `done` | 完了 |
