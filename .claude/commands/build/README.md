# .claude/commands/build

## 1. 概要

`build`フェーズコマンド(`/build:test`・`/build:implement`・`/build:docs`)のClaude Code起動定義を置くディレクトリです。

## 2. 責任

- `test.md`・`implement.md`・`docs.md`: `agent-workflow/commands/build/`の対応ファイルを読む薄いラッパー

## 3. 責任外

- 実行契約の内容そのもの(`../../../agent-workflow/commands/build/`が持ちます)
