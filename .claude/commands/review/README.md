`review`フェーズコマンド(コード・設計・文書・テスト・セキュリティレビュー、監査)のClaude Code起動定義を置くディレクトリです。

## 2. 責任

- `code.md`・`design.md`・`docs.md`・`test.md`・`security.md`・`audit.md`: `agent-workflow/commands/review/`の対応ファイルを読む薄いラッパー

## 3. 責任外

- 実行契約の内容そのもの(`../../../agent-workflow/commands/review/`が持ちます)
