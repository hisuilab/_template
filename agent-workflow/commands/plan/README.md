# agent-workflow/commands/plan

## 1. 概要

要件定義・アーキテクチャ設計・Issue/マイルストーン起票という`plan`フェーズの実行契約を置くディレクトリです。

## 2. 責任

- `requirements.md`: 要件定義の実行契約
- `architecture.md`: アーキテクチャ設計の実行契約
- `design.md`: マイルストーン/Issue設計の実行契約
- `issue.md`: GitHub Issue作成の実行契約
- `milestone.md`: マイルストーンブランチ作成の実行契約

## 3. 責任外

- 要件・設計文書そのものの内容(対象リポジトリの`docs/`が持ちます)
