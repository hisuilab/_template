# agent-workflow/commands/review

## 1. 概要

コードレビュー・設計レビュー・文書レビュー・テストレビュー・セキュリティレビュー・監査という`review`フェーズの実行契約を置くディレクトリです。

## 2. 責任

- `code.md`: 差分のコードレビューの実行契約
- `design.md`: 設計提案レビューの実行契約
- `docs.md`: 文書レビューの実行契約
- `test.md`: テストレビューの実行契約
- `security.md`: セキュリティレビューの実行契約
- `audit.md`: 横断監査(Audit Gate)の実行契約

## 3. 責任外

- レビューの評価軸そのもの(`../../skills/`が持ちます)
